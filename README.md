## 👋 Tentang Repository

Repository ini berisi berbagai file yang saya buat selama mempelajari **PHP (Hypertext Preprocessor)** di kelas **XI RPL Semester 2**.

Seluruh file di dalam repository ini merupakan hasil pembelajaran mulai dari konsep dasar hingga materi yang lebih kompleks, yang dikerjakan selama proses belajar mengajar bersama **Pak Faruq**.

Repository ini juga menjadi dokumentasi perkembangan kemampuan saya dalam memahami bahasa pemrograman PHP, mulai dari:

- 📝 Sintaks dasar PHP
- 🔤 Variabel dan Tipe Data
- ➕ Operator
- 🔀 Percabangan
- 🔁 Perulangan
- 📦 Function
- 🏗️ OOP (Object Oriented Programming)
- ⚡ Static Property & Static Method
- 📂 Studi kasus dan latihan
- 🎯 Tugas kelas
- 🧪 Eksperimen pribadi selama belajar
